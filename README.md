# KtorKMM
 
- Simple REST API With Ktor and show data in  Android App
- I used IntelliJ IDEA 2022.2.2 (Edu) for development

### API Calling

- in cmd type "ipconfig" and copy the address of your Ethernet adapter: IPV4 Address
